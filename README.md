============
Pumba Search
============
Расширение для абонентов компании «Уральские кабельные сети — Озерск» добавляет на популярные киносайты возможность поиска фильма на локальном поисковом сервисе «Пумба».

Поддерживаются следующие сайты:
* kinopoisk.ru
* imdb.com
